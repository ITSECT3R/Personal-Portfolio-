# Common Component Data

This directory contains data files used by common components across the application.

## CV PDF

Place your CV PDF file here with the following naming convention:

- `luis-marin-cv.pdf`

The `useDownloadCV` hook will reference this file for downloads.

## Alternative Locations

You can also place the PDF in:

- `public/data/cv/luis-marin-cv.pdf` - For direct public access
- `src/assets/cv/luis-marin-cv.pdf` - For bundled assets

Choose based on your deployment and caching needs.
